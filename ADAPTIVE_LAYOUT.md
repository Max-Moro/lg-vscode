# Адаптивная верстка Control Panel

## Обзор решения

Реализована полностью адаптивная система верстки для панели управления LG с использованием современных CSS технологий:

### Ключевые технологии
- **Container Queries** - для отзывчивости на основе размера контейнера, а не экрана
- **CSS Flexbox** - для гибкой организации элементов
- **CSS Grid** - не используется для простоты и совместимости
- **Text Overflow Ellipsis** - для усечения длинного текста

### Режимы отображения

#### 1. Обычный режим (>200px)
- Горизонтальное размещение элементов в группах (clusters)
- Автоматический перенос на новую строку при необходимости
- Эластичные размеры с защитой от переполнения

#### 2. Узкий режим (≤200px)  
- **Переключение в одноколоночную компоновку**
- Все элементы растягиваются на полную ширину
- Удаление разделителей (.spacer)
- Радиокнопки остаются горизонтальными для компактности

### Ключевые улучшения

#### Защита от переполнения
```css
/* Полная защита от горизонтального переполнения */
body {
  overflow-x: hidden;
  min-width: 0;
}

/* Текстовое переполнение */
select, button {
  text-overflow: ellipsis;
  white-space: nowrap;
  overflow: hidden;
  max-width: 100%;
}
```

#### Container Queries
```css
/* Основа для адаптивности */
body {
  container-type: inline-size;
  container-name: control-panel;
}

/* Адаптивные правила */
@container control-panel (max-width: 200px) {
  .row {
    flex-direction: column;
    align-items: stretch;
  }
}
```

#### Интеллектуальная группировка
- `.cluster` - логическая группа элементов, которые должны быть рядом
- `.row` - контейнер для горизонтальной компоновки с переносами
- `.spacer` - эластичный разделитель (скрывается в узком режиме)

### Особенности реализации

#### Сохранение логики интерфейса
- Иконки защищены от сжатия через `flex-shrink: 0`
- Лейблы остаются связанными со своими контролами

#### Прогрессивная деградация
- Нормальный режим: полная функциональность
- Узкий режим: компактность без потери функций  
- Экстремальный режим: максимальная экономия места

#### Доступность
- Поддержка `prefers-reduced-motion`
- Улучшенная контрастность для `prefers-contrast: high`
- Правильная фокусировка для клавиатурной навигации

### Тестирование
Решение протестировано на различных ширинах панели:
- ≥400px: комфортное отображение с увеличенными отступами
- 300-400px: стандартное отображение
- 200-300px: компактное отображение с переносами
- < 200px: одноколоночный режим

### Совместимость
- Все современные браузеры с поддержкой Container Queries
- VS Code WebView (основан на Chromium)
- Graceful degradation для старых браузеров

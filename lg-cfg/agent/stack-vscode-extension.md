# Рекомендации в рамках технологического стека

В данном проекте ведется разработка extension для **VS Code**. Это вполне традиционный по структуре проект для этих целей.

**Стек**: TypeScript 5.4, VS Code Extension API ^1.85.0, Node.js 20.11.30

## TypeScript

- Используйте явную типизацию, избегайте `any`
- Импортируйте типы из `vscode` namespace для API расширений
- Enum-ы и union types для состояний (см. `models/`)

## Webview

- Всегда используйте CSP nonce для inline scripts/styles
- Обработка сообщений от webview через typed message handlers
- Для обновления UI после изменения состояния используйте `postMessage` вместо прямых DOM-манипуляций

## Тестирование

Проект не имеет автоматических тестов. Тестирование делается через Extension Development Host (F5) пользователем.

## Разработка UI части блока «Адаптивные возможности» в VS Code Extension

Мы закончили разработку функционального блока «Адаптивные возможности» в CLI интерфейсе и сгенерировали весь необходимый новый API. Теперь необходимо реализовать UI часть для реализации нового функционала.

В панели «LG — CONTROL PANEL» мы ходим создать новый блок, в котором будут расположены контролы для настройки адаптивных возможностей. Данные параметры будут поступать со всем запросами рендеринга контекстов и просмотра статистических отчетов.

Наборы профилей и сами профили мы хотим отображать просто в виде комбобоксов. Наборов профилей не очень много, поэтому их сразу можно расположить в основном блоке для адаптивных возможностей и сразу показать все наборы.

Теги мы хотим включать массово, поэтому хотим располагать в виде наборов чекбоксов и группировать по TAGSETS. Наборов тегов и самих тегов в каждом наборе может быть довольно много, поэтому эту форму лучше спрятать внутрь спойлера или во выезжающий по требованию сайдпейдж.
